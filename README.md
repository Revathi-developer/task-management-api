# Task Management API

A RESTful Task Management API built using Django REST Framework.

## Features

- User Registration
- User Login
- JWT Authentication
- CRUD Operations for Tasks
- Search
- Ordering
- Pagination
- Custom Permissions
- Serializer Validation

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite (Development)
- PostgreSQL (Coming for Production)

## Installation

```bash
git clone https://github.com/Revathi-developer/task-management-api.git

cd task-management-api

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

## API Endpoints

- User Registration
- User Login
- User Profile
- Task CRUD Operations
