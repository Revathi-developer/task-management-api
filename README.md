# Task Management API

A RESTful Task Management API built using Django REST Framework.

## Features

- User Registration
- User Login
- JWT Authentication
- CRUD Operations for Tasks
- Search
- Ordering
- Pagination
- Custom Permissions
- Serializer Validation

## Tech Stack


- python
- Django
- Django REST Framework
- PostgreSQL
- JWT Authentication
- Render

## Installation

```bash
git clone https://github.com/Revathi-developer/task-management-api.git

cd task-management-api

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

- User Registration
- User Login

- Task CRUD Operations
